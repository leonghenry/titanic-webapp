# Titanic Survival Prediction # Version 3 - added suggestions for testing Titanic Survival Prediction

Enter Passenger Details

This web application predicts the survival outcome of a Titanic passenger using two machine learning models: a Decision Tree and a Random Forest. Based on the entered passenger class, sex, age, and fare, each of the models will then estimate whether the passenger would have likely survived or not survived.
